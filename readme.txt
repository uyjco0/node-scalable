

-> The current project is building a demo of a scalable and distributed application composed by Node.js-based web and worker nodes with a PostgreSQL master/slave backend. The scalability and distributed properties are built using Apache Zookeeper, Apache Kafka, Confluent Schema Registry, Confluent Rest Proxy, Confluent Bottledwater Plugin/Client, Node.js, Docker, Weave, Supervisor, HAProxy, Logstash, and PostreSQL with Streaming Replication/Logical Decoding
